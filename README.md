# ☕ Exercícios de Lógica em Java

Repositório dedicado ao desenvolvimento de lógica de programação e consolidação de fundamentos na linguagem Java (estruturas condicionais, laços de repetição, manipulação de strings, arrays e matrizes).

## 📁 Estrutura do Repositório

O repositório está dividido em dois projetos principais:

### 1. 📂 `praticas-complementares`
Projeto focado em desafios de lógica estruturada e manipulação de fluxos básicos, exercícios complementares iniciais.
* **Ex01 - Padronizador de E-mails:** Manipulação de Strings (`.trim()`, `.toLowerCase()`) e uso da classe `Scanner` para tratamento de entradas de dados.
* **Ex02 - Contador de Letras:** Estruturas de repetição aplicadas à varredura de caracteres individuais e higienização de strings (`.replaceAll()`).
* **Ex03 - Carrinho de Compras:** Uso prático de coleções dinâmicas com `ArrayList`, manipulação de métodos como `.add()` e `.size()`.
* **Ex04 - Sistema de Busca:** Implementação de algoritmos de busca em listas utilizando lógica de sinalizadores (`boolean`) e interrupções de laço (`break`).
* **Ex05 & Ex06 - Matrizes (Batalha Naval / Média):** Lógica bidimensional, loops encadeados e controle de escopo de variáveis acumuladoras.

### 2. 📂 `praticas-irede`
Projeto contendo a lista completa de fixação da Unidade 1 (Exercícios de 1 a 10).
* **Ex01 a Ex03:** Operadores lógicos, tipos primitivos e estruturas de menu (`switch-case`).
* **Ex04 a Ex06:** Laços de repetição (`for`), manipulação de Strings, tabuada automática e buscas controladas pelo usuário.
* **Ex07 e Ex08:** Cálculo de fatorial e menus interativos com repetição (`do-while`).
* **Ex09SeparadorNomes:** Quebra de strings dinâmicas usando `.split()` e lógica de vetores (`.length - 1`).
* **Ex10ContadorPalavras:** Contagem automatizada de palavras por espaçamento através de propriedades de Arrays.
