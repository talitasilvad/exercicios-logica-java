# ☕ Exercícios de Lógica em Java

Este repositório contém uma coleção de exercícios práticos desenvolvidos para consolidar os fundamentos da linguagem Java, cobrindo estruturas de repetição, coleções dinâmicas e matrizes.

## 🚀 O que foi praticado:

* **Ex01 - Padronizador de E-mails:** Manipulação de Strings (`.trim()`, `.toLowerCase()`) e uso da classe `Scanner` para tratamento de entradas de dados.
* **Ex02 - Contador de Letras:** Estruturas de repetição aplicadas à varredura de caracteres individuais e higienização de strings (`.replaceAll()`).
* **Ex03 - Carrinho de Compras:** Uso prático de coleções dinâmicas com `ArrayList`, manipulação de métodos como `.add()` e `.size()`.
* **Ex04 - Sistema de Busca:** Implementação de algoritmos de busca em listas utilizando lógica de sinalizadores (`boolean`) e interrupções de laço (`break`).
* **Ex05 & Ex06 - Matrizes (Batalha Naval / Média):** Lógica bidimensional, loops encadeados e controle de escopo de variáveis acumuladoras.
